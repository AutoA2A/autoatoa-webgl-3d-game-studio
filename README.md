# WebGL 3D游戏工坊

## 网站简介
本项目是一个展示WebGL技术的3D游戏工坊网站，通过多个AI协作完成。该网站旨在为用户提供一个体验和探索WebGL技术的平台，展示各种3D游戏和互动场景。

## 页面与功能
网站包含以下6个页面：
1. 首页：介绍网站的主要内容和特色功能。
2. 游戏展示：展示多个基于WebGL技术的3D游戏，包括游戏截图、简介和在线试玩功能。
3. 技术解析：深入解析WebGL技术的原理和应用，包括技术文章和视频教程。
4. 资源下载：提供与WebGL相关的开发资源下载，如代码示例、模型和纹理等。
5. 关于我们：介绍网站的开发团队和联系方式。
6. 联系我们：提供用户反馈和建议的渠道。

## 多 AI 协作与验收
本项目采用多个AI协作完成，包括但不限于：
- 内容生成AI：负责生成网站的文本内容，如游戏简介、技术文章等。
- 图片生成AI：负责生成网站所需的图片，如游戏截图、插图等。
- 代码生成AI：负责生成网站的前端代码，包括HTML、CSS和JavaScript等。
- 测试AI：负责测试网站的功能和兼容性。

为了确保网站的质量和一致性，我们制定了严格的验收标准，包括：
- 内容准确性和一致性
- 页面布局和样式的一致性
- 功能的完整性和正确性
- 兼容性和性能
