API 参考

createScene()

创建一个新的 3D 场景对象，返回 Scene 实例。

const scene = createScene();

loadModel(url)

异步加载 glTF/OBJ 模型，返回 Promise<Mesh>。

loadModel('data:application/octet-stream;base64,...').then(mesh=>scene.add(mesh));

addLight(type, options)

向场景中添加光源，type 可为 "directional"、"point"、"ambient"。

addLight('directional',{direction:[0, -1, 0],color:[1,1,1]});
